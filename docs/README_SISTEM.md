# Dokumentasi Sistem Prediksi Partisipasi Politik (PolPart RF)

Folder ini berisi seluruh rancangan analisis, desain, database, diagram alur, dan dokumen pelaporan akademis (TA/PKL) untuk sistem PolPart RF yang telah diperbarui dengan data level TPS dan sistem autentikasi dua aktor.

## Berkas Dokumentasi:

1. **[FLOWCHART_SISTEM.md](file:///d:/TA/polpart/docs/FLOWCHART_SISTEM.md)**: Diagram alir proses bisnis aplikasi meliputi registrasi, login, penentuan hak akses (role), proses prediksi Random Forest, dan ekspor laporan PDF.
2. **[USE_CASE_DIAGRAM.md](file:///d:/TA/polpart/docs/USE_CASE_DIAGRAM.md)**: Deskripsi aktor sistem (**Admin/Operator KPU** dan **User/Masyarakat**) beserta hak aksenya dalam Diagram Use Case.
3. **[RANCANGAN_DATABASE.md](file:///d:/TA/polpart/docs/RANCANGAN_DATABASE.md)**: Desain skema tabel database SQLite (`polpart.db`) yang telah dimigrasi untuk data tingkat TPS.
4. **[ERD.md](file:///d:/TA/polpart/docs/ERD.md)**: Diagram Entity Relationship (ERD) relasi antar tabel (Mermaid format).
5. **[BAB_III_ANALISA_DAN_DESAIN.md](file:///d:/TA/polpart/docs/BAB_III_ANALISA_DAN_DESAIN.md)**: Draft bab akademis mengenai analisa kebutuhan, perancangan antarmuka, dan permodelan sistem.
